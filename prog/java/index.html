Craig's Java Projects

Java Projects

Here are a few Java projects that I've made. Most of them are available as *.jar files. You'll need the Java JRE to get them to run.

Traffic Simulator

This is a traffic simulator program. It uses:

  • QuadTrees for collision detection.
  • Threading to run each vehicle
  • Graph algorithms for for vehicles to figure out how to get to their goals.

Reverse Polish Notation Calculator

I couldn't find a good reverse polish notation calculator, so I decided to make one. It works pretty well, but it helps if you have a number pad on your computer. It's also got macro recording capability, which is great if you have common sequences of keypresses. For your (my) convenience, this program is also available as an *.exe file for folks on Windows.

Conway's Game of Life

This is a program to run a simulation of Conway's Game of Life. It's super interesting. You can save and load boards and so on.

Drum Machine

Here's a little drum machine that plays through midi. You can save an load loops, but I really need to add the ability to sequence loops together to make a song.

Keyboard Displayer

This is something I made to help my students learn how to type. It turns out that if you can't type without looking at the keyboard, you'll make very slow progress. Lots of my students will type a bit, make a mistake, delete a bit, try to retype what they were going for in the first place, make another mistake, etc. The idea behind this display is that it'll stay on the screen so the students can practice not looking at their hands when they type. It stays on top of all the windows too.

Color Demo

This is just a little program to show how red, green, and blue mix to make colors on monitors, tv, the web, etc.

Number Base Demo

This is something I made to help my students learn about different number bases. Thinking more about, maybe I should show the numbers as graduated cylinders or something where the capacity is the base and the level is the digit.
